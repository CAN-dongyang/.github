## OpenGL
	OpenGL은 범용적인 Graphic을 위해 설계된 API입니다.
	C로 구현되었고 플랫폼에 유연하여
	한때는 표준 Graphics API였으나 현재 업데이트를 중지했습니다.

### Reference Links
| Ref | Link |
|:---:|:---:|
| Learn OpenGL | https://learnopengl.com/Getting-started/OpenGL |
<br>

## Direct3D
	Direct3D는 Microsoft의 Windows 전용 그래픽 라이브러리입니다.

	Native 라이브러리답게 2D, Input, Sound들도 포함할 수 있습니다.

	Microsoft의 UI API, Framework들과 같이 사용하는 것이 일반적입니다.
		Win32, WPF, MFC, WinForm, WinUI 등등

### Reference Links
| Ref | Link |
|:---:|:---:|
| Direct3D 11 Graphic (Win32) | https://learn.microsoft.com/ko-kr/windows/win32/direct3d11/atoc-dx-graphics-direct3d-11 |
<br>

## Vulkan
	Vulkan은 Low-Level Cross-Platform Graphics API입니다.
	OpenGL의 사실상 후속작이며
	고성능 멀티 쓰레딩 애플리케이션에 사용되도록 설계되었습니다.

### Reference Links
| Ref | Link |
|:---:|:---:|
| Khronos Vulkan Tutorial | https://docs.vulkan.org/tutorial/latest/00_Introduction.html |
<br>

## Metal
	Metal은 Low-Level IOS(Apple) Graphic API입니다.
	고수준의 성능과 멀티 쓰레딩을 지원합니다.
	강점이 Vulkan과 같지만 상대적으로 쉬운 난이도라고 합니다.

### Reference Links
| Ref | Link |
|:---:|:---:|
| Metal Tutorial | https://metaltutorial.com |
<br>

	Stardew Valley가 사용한 Framework로 유명합니다.
